### Install MetaMask Embedded Wallets SDK

Install the MetaMask Embedded Wallets SDK (Web3Auth) in your project. Select the tab that applies:

<Tabs
  defaultValue="demo"
  values={[
    { label: 'Demo app', value: 'demo' },
    { label: 'Set up your app', value: 'diy' },
  ]}>
  <TabItem value="demo">

If you cloned the quickstart, install dependencies from the project root:

```bash
flutter pub get
```

  </TabItem>
  <TabItem value="diy">

Use the [Web3Auth Flutter SDK](https://github.com/Web3Auth/web3auth-flutter-sdk) to integrate MetaMask Embedded Wallets into your application. Add `web3auth_flutter` as a dependency to your `pubspec.yaml` file:

```sh
flutter pub add web3auth_flutter
```

or add manually:

```yaml
dependencies:
  web3auth_flutter: ^6.1.1 ### or the latest version
```

  </TabItem>
</Tabs>

:::warning Update to the latest SDK

Use the latest version of the Web3Auth Flutter SDK (for MetaMask Embedded Wallets).
See the [release notes](https://github.com/Web3Auth/web3auth-flutter-sdk/releases).

:::
