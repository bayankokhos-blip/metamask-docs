### Install the MetaMask Embedded Wallets iOS SDK

**Prerequisites**

- iOS 14+
- Xcode 12+
- Swift 5.x

You may follow along with our demo app or configure your own.

**2.1** Clone the quickstart:

```shell npm2yarn
npx degit Web3Auth/web3auth-ios-examples/ios-quick-start w3a-ios-quick-start
```

**2.2** Open the project in Xcode from the `w3a-ios-quick-start` directory and [build the app](https://developer.apple.com/documentation/xcode/building-and-running-an-app).

  </TabItem>
  <TabItem value="diy">

Add the MetaMask Web3Auth iOS SDK to your project:

**2.1** In Xcode, open your project and select **File > Add Packages**.

**2.2** When prompted, add the Web3Auth iOS SDK repository:

   ```bash
   https://github.com/Web3Auth/web3auth-swift-sdk
   ```

**2.3** For **Dependency Rule**, select **Exact Version**, then enter the [latest version](https://github.com/Web3Auth/web3auth-swift-sdk/releases).

Xcode automatically resolves and downloads the dependencies in the background.

  </TabItem>
</Tabs>
